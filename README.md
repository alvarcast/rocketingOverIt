# Rocketing Over It

Inspired by getting over it, help this little robot get to the top of a mountain with their handy rocket launcher.

---

## Project Description
This project was developed for the course 2D Design at Universidad Rey Juan Carlos. The main objective was to apply basic knowledge in design and unity to create a simple game.

### Key Features
* Gameplay Mechanics: The rocket launcher. Propell yourself upwards with precision or fall to the bottom.
* User Interface: Start and pause menus.

---

## Technologies Used
* Game Engine: Unity 6.3.2f1
* Programming Language: C#.
* Integrated Development Environment (IDE): Visual Studio / VS Code.
* Version Control: Git.

---

## Installation and Execution
To view or test the project, please follow the instructions below:

### Running the Build
1. Download the folder located in the `/Build` directory.
2. Run the executable file `trabajoDiseño.exe`.

### Opening the Project in the Editor
1. Clone the repository using the command: `git clone https://github.com/alvarcast/rocketingOverIt`.
2. Launch Unity Hub and add the project from your local folder.
3. Ensure you have the required Unity version installed.
4. Locate and open the main scene at `Assets/Scenes/MainScene.unity`.

Please note that some files had to be ignored due to their size.

You can also open the code with Visual Studio executing the "Rocketing Over It.sln" file
